### **2. Available options** 
- **Encrypt text**: 
- Enter the original text. 
- Specify the encryption key. 
- Output: Encrypted text. 

- **Decrypt text**: 
- Enter the encrypted text. 
- Enter the decryption key. 
- Output: Original text.
